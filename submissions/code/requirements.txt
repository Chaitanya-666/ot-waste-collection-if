# Quick setup
# -----------
# 1. Create and activate a virtual environment
#    python3 -m venv venv
#    source venv/bin/activate   # (Linux / macOS)
#    venv\Scripts\activate      # (Windows)
#
# 2. Install dependencies
#    pip install -r requirements.txt
#
# Running the demo scripts
# ------------------------
# - Basic demo via main.py:
#     python3 main.py --demo basic --save-results --save-plots
#
# - Comprehensive demo (from the demos folder):
#     python3 demos/comprehensive_demo.py --customers 20 --ifs 2 --iterations 500 --save-plots
#
# - Run the test suite:
#     python3 -m unittest discover -s tests -p "test_*.py"
#
# Output
# ------
# - Demo outputs (JSON + PNGs) are written to `demos/outputs/` by the demo script.
# - If you enable `--save-plots`, two PNGs are saved per run:
#     routes_<timestamp>.png
#     convergence_<timestamp>.png
#
# Notes
# -----
# - The repository is designed to run with standard Python 3.10+; if you use a different
#   Python version, you may need to adjust package versions.
# - If plotting fails, ensure `matplotlib` and `pillow` are installed and that an X display
#   (or a headless backend) is available. On headless Linux servers you can set:
#     MPLBACKEND=Agg
#
